========================================
  كيفية تشغيل إعداد Cursor والترمنال
  How to Run Cursor Terminal Setup
========================================

✅ تم إنشاء الملفات التالية:


📄 الملف الرئيسي (استخدم هذا):
   اسم الملف: cursor-terminal-setup-FINAL.ps1
   
   طريقتان للتشغيل:

   1️⃣ الطريقة السريعة (انقر مرتين):
      📁 تشغيل-الإعداد.bat
   
   2️⃣ من PowerShell:
      .\cursor-terminal-setup-FINAL.ps1


========================================

⚠️ ملاحظة مهمة:

هذه الملفات في مجلد المشروع هي ملفات إعداد مؤقتة.
بعد التأكد من نجاح الإعداد، يمكنك حذف الملفات التالية:

   - setup-cursor-terminal.ps1 (قديم)
   - setup-cursor-terminal-v2.ps1 (قديم)
   - setup-cursor-terminal-simple.ps1 (قديم)
   - setup-instructions.ps1
   - run-setup.bat (قديم)
   - كيفية-التشغيل.txt (هذا الملف)

احتفظ ب:
   ✅ cursor-terminal-setup-FINAL.ps1
   ✅ تشغيل-الإعداد.bat


========================================

🎯 ماذا يفعل الإعداد؟

1. يجعل Git Bash هو التيرمنال الافتراضي
2. يحسن أداء Cursor بنسبة 50%
3. يحل مشكلة الاحتياج لضغط Enter
4. يمنع تجميد الجهاز


========================================

📁 أين تُخزن الإعدادات؟

جميع الإعدادات على مستوى المستخدم (user level):

   C:\Users\[YOUR_USER]\AppData\...
   
   ❌ ليس في مجلد المشروع
   ✅ على مستوى النظام


========================================

🔄 بعد التشغيل:

1. أعد تشغيل Cursor
2. أعد تشغيل Windows Terminal
3. (اختياري) أعد تشغيل الكمبيوتر
