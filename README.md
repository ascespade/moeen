# مركز الهمم - Medical Center System

## 🏥 Overview
A comprehensive medical center management system built with Next.js 14, TypeScript, and Supabase.

## ✨ Features
- Patient Management
- Doctor Profiles
- Appointment Scheduling
- Medical Records
- Insurance Claims
- Progress Tracking
- Training Programs
- Admin Dashboard
- AI Chatbot
- CRM System

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Supabase account

### Installation
```bash
npm install
npm run dev
```

### Environment Variables
Create a `.env.local` file with:
```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

## 🧪 Testing
```bash
npm run test
npm run test:e2e
```

## 📦 Build
```bash
npm run build
npm run start
```

## 🚀 Deployment
The system is ready for deployment on Vercel, Netlify, or any Node.js hosting platform.
