## 🎯 مشاريع عملية للتطبيق

### المشروع الأول (الأسبوع 1):
**شات بوت بسيط**
```python
# مثال بسيط
import openai

def chat_bot(question):
    response = openai.Completion.create(
        engine="text-davinci-003",
        prompt=f"أنت مساعد طبي للأطفال. أجب على: {question}",
        max_tokens=150
    )
    return response.choices[0].text
```
